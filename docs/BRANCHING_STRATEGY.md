# Branching Strategy

## Main Branches

### main
Stable production-ready branch.

### develop
Main development integration branch.

## Supporting Branches

### feature/*
New features.

### fix/*
Bug fixes.

### docs/*
Documentation changes.

### release/*
Release preparation.

### hotfix/*
Urgent production fixes.

## Workflow

feature branch
    ↓
develop
    ↓
release
    ↓
main
